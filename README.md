# 两学刷题

广东省高校教师资格两学考试（高等教育学、高等教育心理学）备考刷题工具。

## 功能

- **题库**：973 道选择题（高等教育学 472 题 + 高等教育心理学 501 题），涵盖单选、多选、判断
- **科目/题型筛选**：支持按科目和题型自由组合出题
- **随机/顺序模式**：随机抽题或按题库顺序刷题，数量可调
- **错题集**：自动收集错题，支持筛选、移除、专项练习
- **收藏**：刷题中收藏重点题目，方便回顾
- **刷题记录**：每题答题统计，按状态筛选
- **离线可用**：纯前端，数据存储在 localStorage

## 运行

```bash
python3 -m http.server 8080
# 浏览器打开 http://localhost:8080
```

## 部署

推送到 `master` 分支自动部署到 GitHub Pages。
