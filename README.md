# 两学刷题

广东省高校教师资格两学考试（高等教育学、高等教育心理学）备考刷题工具。纯前端，离线可用。

## 题库

973 道选择题，覆盖高等教育学（472 题）和高等教育心理学（501 题），包含三种题型：

- **单选** — 四选一
- **多选** — 全对得分
- **判断** — 正确/错误

## 出题模式

- **随机出题** — 自定义数量，每次随机抽取
- **顺序出题** — 按题库编号依次刷题，自动保存进度，支持续刷
- **跳过已答对** — 开启后自动过滤已掌握的题目，专注薄弱点

## 科目与题型筛选

按科目（全部 / 高等教育学 / 高等教育心理学）和题型自由组合，实时显示符合条件的题量。

## 错题集

答错的题目自动收录，支持：

- 按科目筛选
- 单题移除或一键清空
- 错题专项练习，做对了自动移出

旧版本数据自动迁移兼容。

## 收藏

刷题中收藏重点题目，方便考前回顾。同样支持专项练习。

## 刷题记录

每道题的作答次数和正确率终身记录，支持按科目、状态（已刷/未刷）筛选，点击任意题目可单题重刷。

## 设置持久化

科目选择、题型筛选、出题模式、跳过已答对开关等设置自动保存，下次打开恢复上次状态。
